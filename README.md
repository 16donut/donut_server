
# donut_server

### 1. 제공 서비스
- 전자 처방전을 활용한 사용자 간편 복약 관리 플랫폼
- 불용의약품 수거 서비스
- 전자 처방전을 이용한 의약품 시장


### 2. 핵심기능
- 복약관리
- 캘린더
- QR코드 복약정보 등록
- 불용의약품 수거 알림
- 주변 약국 안내
